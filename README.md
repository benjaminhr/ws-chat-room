# chatroom

## specs: 
  - backend:
    - python and node versions of:
      - websocket server
      - https server
    - containerised with docker
    - in kuberenetes cluster 
    - deployed via GCP

  - frontend:
    - react
    - deployed to netlify 

  - to run: 
    - `npm i && npm start:node` or `npm i && npm start:python`

    - backend can be run using docker
